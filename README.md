Contributing
---------
Contributions are welcome.
In your console or equivalent:
```
$ git clone https://github.com/danielstpaul/edstpaul_website.git [folder name]
$ cd [folder name]
$ cd app
$ http-server
```
Then simply create a pull request when you have made any changes!
